# Books and Quotes Front-end
This is the front-end from a web application to handle CRUD functionality for a list of books and quotations.

## Features
- Responsive
- Collapsable menu
- Dark mode
- Font Awesome Icons

## Technologies
- Angular
- Bootstrap
- Typescript

## Available 
The app is available for showcase at: https://red-code-frontend.vercel.app/login

## Backend
The backend of this app was made with C# and is available at; https://github.com/albertito1985/Red_Code_backend
